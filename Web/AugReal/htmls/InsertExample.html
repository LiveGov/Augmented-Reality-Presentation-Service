How to insert a new entity

Once you have the rights to create an Entity, press the 'Add new' button. A new Entity is added on the top of the list. By pressing 'Edit' you are able to add text and multimedia information to the Entity. The following information should be gathered before inserting a new entity:
  1. Title and Description of your entity. Multilingual support. Apostrophe ' should be avoided.
    -Title: A new bench
    -Description: This is a ...
  2. Images necessary. Certain images should be gathered.
    -Background: Image of the issue (JPG, size about 640 x 480).
    -3D model image: Screenshot of the 3D model (JPG, size about 640 x 480).
    -Tracking pattern: Image for tracking the issue (PNG, exact size 200 x 200 black and white). Bigger images cause higher delays.
  3. 3D model. It consists of three files.
    -Coordinates: OBJ format. Example:
    -Material: MTL format.
    -Textures: JPG format. All textures should be in jpg format inside an images folder.

F.A.Q.:

  1. The 3d object has no color?
    Use blender tool to export to obj file with Normals.
  2. My model is not in obj format.
    - For Collada DAE or 3D studio 3ds formats use blender tool -> export to obj.
    - For Autodesk FBX, use free converter by Autodesk.
